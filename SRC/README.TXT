
        1. Be sure to unzip this file with the -d option.

        2. Create the following directory structure:

           \WHAVEN2\SRC
           \WHAVEN2\RUN

           Install the Witchaven II game files in the RUN directory and unzip
           the WH2SRC file in the SRC directory.

        3. To compile Witchaven II execute the WH.BAT file.  Be sure all .DEF
           files are up-to-date.  Run PROTO.BAT to update the .DEF files.

           To compile:

           C:\WHAVEN2\SRC> wh

           To update .DEF files:

           C:\WHAVEN2\SRC> proto
